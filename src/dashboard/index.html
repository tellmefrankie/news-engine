How to Register a Skill

1. Go to agensi.io → List a Skill
2. Upload ZIP (drag & drop)
3. Fill in Skill Name, Summary, Description (auto-generated from SKILL.md)
4. Add Tags, Keywords, Use Cases
5. Set price (Free or Paid)
6. Submit → Security review → Published
